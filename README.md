# AURA — The Ambient Focus OS1

A premium, ultra-modern, and fully responsive landing page for **AURA**, a futuristic productivity software designed to induce deep cognitive flow by orchestrating personalized spatial acoustics and circadian workspace light sync.

Built with **HTML5, Vanilla CSS3, and lightweight modern JavaScript**.

---

## 🌟 Visual & Interactive Features

- **Drifting Ambient Blobs**: Highly optimized background blur shapes utilizing CSS keyframes to establish a deep, sleek space aesthetic.
- **Glassmorphic UI Cards**: Translucent interface surfaces employing modern CSS backdrop-filters and fine border glows.
- **Interactive Sandbox & Dynamic Themes**: A live dashboard widget allowing users to click through 4 curated focus modes:
  - 💜 **Indigo Flow**: Curved gradients matching `8.2 Hz Alpha` waves to stabilize cognitive loads.
  - 💚 **Tranquil Mint**: Emerald-cyan ambient lighting (`11.5 Hz Alpha Flow`) to reduce visual eye-strain.
  - 🧡 **Cyber Pink**: High-energy orange-pink grids (`14.2 Hz Beta Creative`) for prototyping and rapid design sprints.
  - 💛 **Solar Amber**: Calming amber gradients (`6.5 Hz Theta Calm`) to prompt deep workspace decompression.
- **Vector Mockup Dashboard**: A fully HTML/CSS-drawn interactive application mockup displaying simulated soundwaves, circadian charts, and live load percentages that update instantly when the site theme is changed.
- **Eased Statistics counters**: Key stats count up smoothly from zero on page load using a professional **cubic-deceleration curve** (`ease-out`).
- **Responsive Mobile Navigation Drawer**: Mobile view collapses cleanly to include a custom interactive slide-in burger menu.

## 🛠️ Technology Stack

- **HTML5**: Semantic tags, responsive layout structures, and accessibility attributes.
- **CSS3**: Custom property (variables) themes, Flexbox grids, media query breakpoints, and hardware-accelerated transitions.
- **JavaScript**: Modular, lightweight ES6 scripting for animation states, scroll listeners, and form validations.

## 🚀 How to Run Locally

Since this is a front-end project with zero external dependencies, running it is simple:

### Option A: Local Double-Click
Simply open **`index.html`** in any web browser of your choice.

### Option B: Local Web Server (Recommended)
Starting a local server handles relative file requests perfectly:

**Using Python:**
```bash
python -m http.server 8000
```
Then visit: `http://localhost:8000`

**Using Node.js:**
```bash
npx http-server
```
Then visit the local address output in your terminal.

## 📱 Responsiveness Configuration

The system transitions smoothly across three critical breakpoint definitions:
1. **Desktop (`>= 1024px`)**: Split layouts, isometric 3D visual rotations, full-row footers.
2. **Tablet (`768px - 1023px`)**: Reflows columns to single central columns, flattens perspectives, refactors grid grids.
3. **Mobile (`< 767px`)**: Burger nav menu drawers, compact font ratios, blocks inputs vertically for quick thumb usage.
